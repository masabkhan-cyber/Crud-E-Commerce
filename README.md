A Basic Crud App For Demonstration
Intergrated with: 
Admin LTE,
Bootstrap,
DataTables



prerequisite 

1) Install Xampp
2) Install Composer
3) Install Nodejs




Installation Method




git clone https://github.com/masabkhan-cyber/Crud-E-Commerce.git

cd your-laravel-app

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

php artisan storage:link

npm run dev

php artisan serve
